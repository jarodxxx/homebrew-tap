# homebrew-tap

Homebrew tap for [Claudex](https://github.com/jarodxxx/Claudex) and other tools.

## Install Claudex

```bash
brew install --cask jarodxxx/tap/claudex
```

## Available casks

| Cask | Description |
|---|---|
| [claudex](Casks/claudex.rb) | macOS menu bar app aggregating Claude.ai, RTK and MemPalace usage stats |
